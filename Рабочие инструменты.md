# Рабочие инструменты

Вот наши основные рабочие инструменты.  

- Корпоративный мессенджер — [Teams](https://www.microsoft.com/ru-ru/microsoft-teams/group-chat-software)
- Трекеры задач — [Jira](https://www.atlassian.com/software/jira)
- База знаний и документация — [Confluence](https://www.atlassian.com/software/confluence)
- VCS — Git ([GitLab](https://about.gitlab.com/))
- CI — [Jenkins](https://www.jenkins.io/)
- Совместная работа - [Miro](https://miro.com)
